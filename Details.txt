Blazor decouples the business logic from the UI.
Blazor server hosts Razor components on the server in ASP.NET Core.
Telerik UI for Blazor is used for represantig the data from database. The data is visualized by 
Grid component. It provides easy paging, sorting, filtering.

This project is created following the instructions from official documentation
here: https://docs.telerik.com/blazor-ui/getting-started/server-blazor

Grid components : https://demos.telerik.com/blazor-ui/grid/overview
